markMEIZAN主催のIncubation Program Spread Pt2にて最優秀賞を受賞しました

鹿児島市の施設であるmarkMEIZANで主催されたIncubation Program Spread Pt2にて谷村、伊地知、松田で最優秀賞を受賞することができました。
受賞したサービスはMEFARのメインサービスであるドコロカです。ドコロカは空きテナントを活用した荷物を預けることのできるサービスです。
審査員であるGMOペパボ代表の佐藤 健太郎氏は「空きテナントという社会課題をインバウンドを使って解決しようという発想が素晴らしいと思いました。この問題は鹿児島に限らず、他の地域でも同様に発生しているため、スケール化できる可能性もあります。今後も期待しています！」とコメントしていただきました。
